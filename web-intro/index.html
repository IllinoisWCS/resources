The Building Blocks of Websites

What is HTML?

  • Hypertext Markup Language
  • A scripting language - interpreted line-by-line by a web browser to render different elements
  • Makes use of starting and ending tags
  • The most recent version is HTML5

What is CSS?

  • Cascading Style Sheet
  • Provides web developers with ability to customize how their websites look
  • You can change any attribute of any HTML element.
  • Examples: fonts, colors, border, measurements
  • The most recent version is CSS3

What is Javascript?

  • Provides web developers with ability to do interactions and animations on their websites
  • Most commonly used to react to user interactions on the website
  • Many frameworks that exist built on top of this language
  • Not at ALL like Java (despite the name)

How powerful are these tools?

  • Just HTML - minimal CSS
  • HTML, CSS only
  • HTML, CSS, JS

Part 4: Navigation Bar

Let’s make a navigation bar with all of your webpages. Notice that we’re using an unordered list. Because we’re going to be using CSS to change its appearance, we’re going to surround the list with a <div> element and give the navbar a class name called “navbar”.


	<div class="navbar">
	    <ul>
	        <li><a href="index.html">Home Page</a></li>
	        <li><a href="favbooks.html">Favorite Books</a></li>
	        <li><a href="favmovies.html">Favorite Movies</a></li>
	        <li><a href="favplaces.html">Favorite Places</a></li>
	        <li><a href="favfood.html">Favorite Foods</a></li>
	    </ul>
	</div>

CSS Code Snippets

In EVERY html document, you have to provide a link to your style sheet so that your html pages can access the customizations you put in styles.css.

Within your head tag, place the following code:


	<link rel="stylesheet" type="text/css" href="styles.css" />
  • rel - gives the category of the link
  • type - gives the language used in the document
  • href - gives the location of the document relative to the current directory

Part 1: Background

If you want to change the entire background of all your web pages, here’s how. Give your body a class within your HTML (<body class=”bcenter”>). You can even change the width and have your web page centered.


	.bcenter{
	    background-color: black;OR
	    background-image: url(‘ image name ‘) no-repeat center center fixed;
	    max-width: 960px;
	    margin: 0 auto;
	}

Part 2: Text

Within any <h#> tag (# 1 through 6), or even <p> tags, you can change the font type, color, style, text alignment, anything!


	h1{
	    font-family: font name like Times New Roman;
	    color: (color like red, black, blue or use a hex code);
	    text-align: (center, left, or right);
	    font-size: #px;
	    text-transform: (uppercase, lowercase, etc)
	}

Part 3: Navigation Bar

Let’s style our navbar now by using the class name we gave it. Because there are two elements within our class, <ul> and <li>, we need to style each one separately. We can also style the <div> itself.


	.navbar{
	    float: (center, right, left);
	    padding: #px;
	    display: block;
	    width: #px;
	    background-color: color;
	    border-radius: #px;
	}

Part 3: Navigation Bar Continued

Let’s style the <ul> elements. We need to make sure that the list elements do not have bullets next to them, nor do they just sit on top of one another. We also need to make sure that the elements don’t run longer than the width we set for our website.


	.navbar ul{
	    list-style-type: none;
	    margin: #px #px #px #px; (top right bottom left)
	    padding: #px #px #px #px; (top right bottom left)
	    overflow: hidden;
	}

Part 3: Navigation Bar Continued

Now, let’s style the <li> elements. We want to them to show up all in one line instead of on top of each other (actually, it’s your choice, really).


	.navbar li{
	    display: inline;
	    padding: #px #px #px #px; (top right bottom left)
	    font-family: (Times New Roman)
	    font-style: (bold, italic, etc.)
	    font-size: #px
	    text-transform: (uppercase, lowercase, etc.)
	    text-decoration: (underline, line-through, etc.)
	}

Part 4: Images


	img{
	    width: #px;
	    height: #px;
	    border-radius: #px; (to produce rounded corners)
	    box-shadow: #px; (to produce a shadow behind it)
	    border-style: See handout
	    margin: #px #px #px #px; (top right bottom left)
	    padding: #px #px #px #px; (top right bottom left)
	}

Other Tidbits

  • You can style the entire table itself (like giving it a background color or a set width, height etc), table rows, or table cells.
  • There are multiple states for links (especially in the navigation bar): hover, active, visited. You can customize each.
  • Say you want to style certain cells but not others. If you want to style multiple cells with the same style, add the class attribute to the <td> element in your HTML. If you just want to style a specific cell, add the id attribute.
